# Contextus
### Adaptive AI-Powered Text Corrector

> *"A word alone means nothing. Meaning comes from what came before."*

Contextus is an AI-powered text corrector that adapts to each user's writing style, emotional context, and conversation patterns; going far beyond dictionary-based correction by understanding meaning, habit, and context.

---

## The Problem

Traditional autocorrectors treat deviations from standard language as errors.  
But "vc", "tb", and "pq" are not mistakes, they are patterns.  
A corrector that doesn't understand the difference between an error and a habit will always get in the way.

Beyond vocabulary, current correctors ignore:
- Who the user is talking to
- What time of day it is
- The emotional tone of the conversation
- The user's personal communication style built over time

**Contextus is built to solve exactly that.**
